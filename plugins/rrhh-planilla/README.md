# rrhh-planilla

Plugin de validación y comparación de planillas para el equipo RRHH Taxtic.

## Componentes

- **Skill `/validar-planilla`** — chequea estructura, aritmética, descuentos legales.
- **Skill `/comparar-planillas`** — mes vs mes anterior con umbral configurable.

## Instalación

```
/plugin install rrhh-planilla@plugins-taxtic
```
